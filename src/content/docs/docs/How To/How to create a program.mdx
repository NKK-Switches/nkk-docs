| Parameter | Input description                                                        |
| --------- | ------------------------------------------------------------------------ |
| Name      | The human readable comment of the input.                                 |
| Pin       | The pin number that the input will represent.                            |
| Type      | The type of input. Either analog or digital.                             |
| Pull      | Either pull up or pull down. If no pull is needed, use DNP (Do Not Pull) |
